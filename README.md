# Personal Finance Tracker (Kişisel Finans Takip Sistemi)

Bu proje, Işık Üniversitesi Bilgisayar Mühendisliği 1. sınıf eğitimi kapsamında Java kullanılarak geliştirilmiştir.

## Projenin Amacı
Kullanıcıların gelir-gider takibini yapabilmesi, bütçelerini mevsimsel veya aylık olarak yönetebilmesi ve harcama limitlerini kontrol altında tutması amaçlanmıştır.

## Teknik Detaylar
* **Dil:** Java (SE)
* **Kavramlar:** Switch-Case Structures, Loops (While), String Manipulation, Basic Algorithms.

## Özellikler
* **Dinamik Bütçe Hesaplama:** Kira, fatura ve market masrafları seçilen periyoda (haftalık/aylık/sezonluk) göre otomatik hesaplanır.
* **Mevsimsel Değişkenlik:** Doğalgaz faturası yaz aylarında düşer, kış aylarında artar.
* **Limit Kontrolü:** Eğlence (Entertainment) harcamaları için aylık 9000 TL limit kontrolü vardır.
* **Yatırım Simülasyonu:** Rastgele (Random) yatırım geliri oluşturma özelliği.
